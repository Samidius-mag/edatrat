# JSON Хранилище данных

Эта директория содержит все данные приложения в формате JSON.

## Структура файлов

### Начальные данные (в репозитории)
- `products.json` - база продуктов
- `recipes.json` - база рецептов
- `seasonal_products.json` - данные о сезонности

### Пользовательские данные (не в репозитории)
- `users.json` - пользователи (создается автоматически)
- `user_profiles.json` - профили пользователей
- `pantry_items.json` - остатки в холодильнике
- `weekly_menus.json` - недельные меню
- `shopping_lists.json` - списки покупок
- `store_discounts.json` - акции магазинов

## Формат данных

Каждый файл содержит массив объектов JSON. Каждый объект должен иметь уникальный `id`.

Пример:
```json
[
  {
    "id": 1,
    "name": "Пример",
    "created_at": "2024-01-01T00:00:00.000Z"
  }
]
```

## Автоматическое создание

Если файл не существует, он будет создан автоматически при первом обращении.
